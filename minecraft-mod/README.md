# EternityRaces — NeoForge 1.21.1 Minecraft Mod

**Мод ID:** `racecraft`  
**Пакет:** `com.eternity.races`  
**Версия Minecraft:** 1.21.1  
**Загрузчик:** NeoForge 21.1.x  
**Язык:** Весь игровой текст на **русском** языке.

---

## Описание

**EternityRaces** добавляет в Minecraft систему из **15 уникальных рас**, каждая с:
- **8 активными способностями** (клавиши R, F, C, V, B, N, M, ,)
- **Пассивными чертами** и **штрафами**
- **Деревом прогрессии** из 3 уровней и 2 веток

Раса выбирается один раз при первом входе (книга «Том рас») и **не может быть изменена**.

## Дерево способностей

```
[Tier 1: Способность 0] [Способность 1] [Способность 2]  ← бесплатно
         ┌──────────────────────┬──────────────────────┐
     [Ветка A]               [Ветка B]
  [Сп. 3] [Сп. 4]         [Сп. 5] [Сп. 6]              ← 20 очков/шт
                    [Tier 3: Способность 7]              ← 30 очков
```

Очки рас зарабатываются за убийство мобов.

## Управление (по умолчанию)

| Клавиша | Действие |
|---------|---------|
| **G** | Открыть дерево способностей |
| **R** | Способность 1 (Tier 1) |
| **F** | Способность 2 (Tier 1) |
| **C** | Способность 3 (Tier 1) |
| **V** | Способность 4 (Ветка A) |
| **B** | Способность 5 (Ветка A) |
| **N** | Способность 6 (Ветка B) |
| **M** | Способность 7 (Ветка B) |
| **,** | Способность 8 (Tier 3) |

## Конфигурация

Файл: `config/racecraft-common.toml`

| Параметр | Описание | По умолчанию |
|---------|---------|-------------|
| `cooldownMultiplier` | Множитель кулдауна | 1.0 |
| `damageMultiplier` | Множитель урона | 1.0 |
| `pointGainMultiplier` | Множитель очков | 1.0 |
| `grantBookOnJoin` | Книга при входе | true |
